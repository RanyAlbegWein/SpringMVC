# Stage 5

Add the following dependency to *pom.xml*

```xml
       <dependency>
            <groupId>javax.servlet</groupId>
            <artifactId>jstl</artifactId>
            <version>1.2</version>
        </dependency>
```

Tab library:

```xml
<%@ taglib uri="http://java.sun.com/jsp/jstl/core" prefix="c"%>
```
